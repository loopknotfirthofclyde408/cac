## v1.3.0
<sub>2026-03-26</sub>

**Security enhancements** ‚Äî [PR #28](https://github.com/nmhjklnm/cac/pull/28) / [Issue #27](https://github.com/nmhjklnm/cac/issues/27)

Based on reverse engineering of Claude Code 2.1.83, fixed multiple fingerprint collection points:

- **anonymousId**: A second persistent identifier in `.claude.json` (format: `claudecode.v1.<uuid>`) ‚Äî now auto-reset on environment switch
- **statsig.stable_id**: Fixed silent failure when no existing file; creates fallback `statsig.stable_id.local`
- **GrowthBook**: New A/B testing framework discovered ‚Äî `cdn.growthbook.io` added to DNS block list
- **Per-interface MAC spoofing**: Different network interfaces now get different derived MACs (XOR with interface index), avoiding the "all interfaces share identical MAC" anomaly
- **Stale field cleanup**: `numStartups`, `firstStartTime`, `cachedGrowthBookFeatures`, `cachedStatsigGates` cleared on environment switch

**Streaming `cac env check`** ‚Äî [PR #28](https://github.com/nmhjklnm/cac/pull/28)

- Each check result displays immediately ‚Äî no more 20-second blank wait
- Animated dots during IP detection
- Multi-source exit IP detection with fast retry: api.ip.sb, ip.3322.net, ipify, ipinfo
- TUN conflict detection with relay bypass status

**CI/CD**

- Prerelease versions (e.g. `1.3.1-beta.1`) auto-publish to npm `beta` tag

## v1.0.0
<sub>2026-03-23</sub>

**Initial architecture** ‚Äî [PR #20](https://github.com/nmhjklnm/cac/pull/20)

Rewrite from proxy management script to uv-style Claude Code manager:

- **Version management**: `cac claude install/ls/pin/uninstall`
- **Environment isolation**: `cac env create/ls/rm` with per-environment `CLAUDE_CONFIG_DIR`
- **Node.js fingerprint hook**: Intercepts `os.hostname()`, `os.networkInterfaces()`, `child_process.execSync('wmic/ioreg')` ‚Äî [PR #5](https://github.com/nmhjklnm/cac/pull/5)
- **DNS guard**: Blocks telemetry domains via `dns.lookup` + `net.connect` + `fetch` monkey-patching ‚Äî [PR #3](https://github.com/nmhjklnm/cac/pull/3)
- **Health check bypass**: In-process Node.js interception, no `/etc/hosts` or root needed
- **Local relay**: Auto-bypasses TUN interfaces via loopback forwarding
- **mTLS**: Self-signed CA + per-environment client certificates
- **Docker mode**: sing-box TUN network isolation + cac identity protection ‚Äî [PR #8](https://github.com/nmhjklnm/cac/pull/8)
